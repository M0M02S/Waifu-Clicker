# Kaskaden-Pop
Stell dir ein Raster vor (z.B. 8x8), gefüllt mit bunten Kugeln oder Blöcken in verschiedenen Farben. Du tippst auf eine zusammenhängende Gruppe von mindestens zwei gleichfarbigen Elementen – und die gesamte Gruppe explodiert! Die darüber liegenden Teile fallen nach unten, und von oben werden neue, zufällige Farben nachgeschoben.
